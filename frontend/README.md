# Pobeda Frontend

Клиентская часть приложения для бронирования объектов на React.

## 📋 Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── Navbar.tsx
│   ├── Layout.tsx
│   ├── ObjectCard.tsx
│   ├── LoadingSpinner.tsx
│   ├── ErrorMessage.tsx
│   ├── LoginForm.tsx
│   ├── RegisterForm.tsx
│   └── ProtectedRoute.tsx
├── pages/              # Страницы приложения
│   ├── HomePage.tsx
│   ├── LoginPage.tsx
│   ├── RegisterPage.tsx
│   ├── ObjectsPage.tsx
│   ├── ObjectDetailPage.tsx
│   ├── ReservationsPage.tsx
│   ├── ReservationDetailPage.tsx
│   ├── ProfilePage.tsx
│   └── MenuPage.tsx
├── services/          # API клиент
│   └── api.ts
├── hooks/            # Кастомные хуки
│   ├── useAuth.ts
│   └── useApi.ts
├── store/            # State management (Zustand)
│   └── authStore.ts
├── types/            # TypeScript типы
│   └── index.ts
├── utils/            # Утилиты
│   └── formatters.ts
├── styles/           # CSS стили
│   └── globals.css
├── App.tsx           # Главный компонент с маршрутизацией
└── main.tsx          # Entry point
```

## 🚀 Установка и запуск

### Требования

- Node.js 18+
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Приложение будет доступно на `http://localhost:5173`

### Сборка

```bash
npm run build
```

## 🛠️ Используемые технологии

- **React 18** - библиотека для создания UI
- **TypeScript** - типизация JavaScript
- **React Router** - маршрутизация
- **TanStack Query** - управление состоянием сервера и кеширование
- **Zustand** - легковесное управление состоянием
- **React Hook Form** - управление формами
- **Zod** - валидация схем
- **Tailwind CSS** - утилиты CSS
- **Axios** - HTTP клиент
- **Vite** - сборщик проекта

## 🔐 Безопасность

- Токены хранятся в localStorage
- Интерцепторы для добавления токена в заголовки
- Автоматический редирект при истечении токена
- Валидация форм на клиенте и сервере

## 🎨 Дизайн

- Responsive дизайн (мобильный, планшет, десктоп)
- Tailwind CSS для быстрого стилирования
- Консистентная цветовая схема
- Удобный интерфейс пользователя

## 🔄 State Management

### Zustand для аутентификации

- Хранение информации о пользователе
- Управление токенами
- Синхронизация с localStorage

### TanStack Query для API

- Кеширование данных
- Автоматическое обновление
- Обработка ошибок
- Оптимистичные обновления

## 📚 API интеграция

API клиент автоматически:

- Добавляет токен в заголовки
- Обрабатывает ошибки 401
- Преобразует данные в нужный формат
- Обеспечивает интерцептирование запросов

## 🧪 Валидация

Используется Zod для валидации:

- Форма логина
- Форма регистрации
- Формы бронирования

## 📱 Адаптивность

Все компоненты адаптированы для:

- Мобильных устройств (320px+)
- Планшетов (768px+)
- Десктопов (1024px+)

## 🚦 Состояния компонентов

- Loading - отображение спиннера
- Error - отображение ошибки с возможностью повтора
- Empty - пустое состояние
- Success - успешное состояние
